# To-Do List Project

To put my Front-End studies into practice, I decided to build a simple to-do list project using HTML, CSS, and JavaScript.

The website is not responsive yet, but I will implement this feature soon.

I used VS Code as my text editor to kick off the project. I also used Gemini and Claude for learning purposes, which helped me immensely in finishing this first version.

This project is strictly for learning purposes, but if it helps anyone, it has already done its job.
